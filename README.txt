With Scopa you can show interactive tags on your content to drive high-intent traffic to your desired destination, resulting in greater engagement and tailored analytics. _For more information on shoppable product tagging please visit our [website.](https://www.scopaanalytics.com/)_

== Installation ==

= Automatic Installation =

1. Go to **Plugins -> Add New**.
2. Search for **Scopa Script**.
3. Install and activate the Scopa Script plugin.

= Manual Installation =

1. Download the latest tagged archive.
2. Go to **Plugins -> Add New** 
3. Click on **Upload Plugin**.
4. Upload the zipped archive directly.
5. Activate the Scopa Script plugin.
